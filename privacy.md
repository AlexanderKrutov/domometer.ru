# Политика конфиденциальности

## Список разрешений

<div markdown="1" class="instruction">
Приложение DomoMeter *(возможные наименования далее по тексту: Программа, Приложение)* обладает большим набором различных функций, некоторые из которых, согласно архитектуре операционной системы Android, требуют особых разрешений от пользователя. Ниже перечислен полный список разрешений, которые может запрашивать программа:

* **Считывание содержимого SD-карты, Изменение или удаление содержимого SD-карты** 
— требуется для хранения данных Программы, таких как списки счётчиков и услуг, история показаний счётчиков, история платежей, а также для хранения прикрепленных фото квитанций. 
Эти разрешения являются обязательными для работы Программы.

* **Проверка лицензии Google Play** — требуется для проверки возможности запуска Программы для пользовательского аккаунта Google. Является обязательным только для полной версии Программы.

* **Фото- и видеосъемка** — требуется для доступа к камере устройства. Доступ к камере необходим для создания фотографий квитанций счетов. Это разрешение не является обязательным.

* **Просмотр контактов** — требуется для доступа к адресной книге устройства. Доступ к контактам необходим для выбора телефонного номера или адреса электронной почты при отправке показаний счётчиков. Это разрешение не является обязательным.

* **Поиск аккаунтов на устройстве** — требуется для доступа к облачным хранилищам с использованием существующих аккаунтов облачных сервисов. Необходимо для корректной работы резервной копии по расписанию в облако. Это разрешение не является обязательным.

* **Доступ в интернет** — требуется для доступа к облачным хранилищам. Необходимо, чтобы Программа могла загружать файлы резервной копии в облако, если настроено расписание. Это разрешение не является обязательным.

* **Управление вспышкой** — требуется для использования вспышки камеры (при её аппаратном наличии) в качестве фонарика для подсветки при снятии показаний счётчиков. Это разрешение не является обязательным.

* **Запуск при включении устройства, Отключение спящего режима** — необходимы для работы напоминаний об оплате коммунальных услуг и снятии показаний. Эти разрешения не являются обязательными.

## Сбор, хранение и обработка конфиденциальных данных

### Какие персональных данные использует Программа

Программа не собирает никаких дополнительных данных, кроме тех, что явно указывает пользователь при создании счётчиков, услуг, счетов, платежей, внесении показаний и прочем взаимодействии с Приложением в рамках заявленного функционала.

Среди указанных пользователем данных могут такие персональные данные как Фамилия, Имя, Отчество (могут использоваться при создании QR-кода для оплаты счетов через платёжные терминалы), адрес или адреса проживания (могут указываться пользователем в свойствах квартир), номера и реквизиты лицевых счетов для оплаты.
Кроме того, персональные данные могут присутствовать на фотографиях квитанций об оплате.
Любое указание этих и других данных происходит только путём внесения этих данных самим пользователем. Указание этих или иных персональных данных не является требованием к работе программы и происходит только по инициативе пользователя.

### Где хранятся данные программы

Программа ни в каком виде не собирает, не использует, не хранит на внешних серверах и не пересылает информацию в интересах разработчика или иных (третьих) лиц. 
Все данные Приложения хранятся исключительно на устройстве пользователя или в его персональном облачном пространстве (Google Drive и/или Dropbox — только при условии настроенной резервной копии по расписанию). 
Ни автор приложения, ни прочие лица не имеют доступа к персональной базе данных пользователя.

### Финансовые операции и взаимодействие с денежными средствами и счетами

Программа не осуществляет никаких финансовых операций, а предназначена только для учёта и контроля расходов в коммунальной сфере.
Все денежные суммы, указанные в программе, являются лишь числовой информацией и никак не связаны с реальными денежными суммами и финансовыми средствами.
Программа не имеет доступа к реальным финансовым счетам и не осуществляет платежи и иные операции по этим счетам.
